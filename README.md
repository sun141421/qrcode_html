# 📱 QR Code 生成器

一个纯前端、零依赖的 QR Code 生成器，**单文件 HTML**，浏览器中直接运行。

基于 **ISO/IEC 18004** 标准从头实现 QR 码编码引擎，无需任何外部库、CDN 或服务端支持。

---

## 功能特性

- **零依赖** — 纯 HTML+CSS+JS，一个文件搞定，无需安装或加载任何外部资源
- **完整 QR 码标准支持** — 版本 1-40，4 种纠错级别（L/M/Q/H）
- **支持中文 & 多语言** — UTF-8 编码，中文、日文、特殊字符均可
- **自定义配色** — 前景色、背景色自由选择
- **尺寸调节** — 200px ~ 500px 可选
- **PNG 下载** — 一键导出高清二维码图片（2x 缩放）

## 如何使用

直接在浏览器中打开 `index.html`：

```bash
# Windows
start index.html

# macOS
open index.html
```

无需任何构建步骤或服务端。

### 操作

1. 输入任意文本、URL、数字等内容
2. 选择尺寸和纠错级别
3. 可选调整前/背景色
4. 点击 **生成二维码** 或按 `Enter` 键
5. 点击 **下载 PNG** 保存

## 技术架构

```
index.html
├── CSS (响应式卡片布局, 渐变色背景)
├── QR 码编码引擎 (自实现)
│   ├── GF(256) 有限域运算 — 指数/对数查表
│   ├── Reed-Solomon 纠错编码 — 多项式生成器 + 长除
│   ├── 数据编码 — 数字(10bit/3位) / 字母(11bit/2字符) / 字节(UTF-8)
│   ├── 版本选择 — 根据 ECC 级别精确计算位容量
│   ├── 矩阵构建 — 定位图案、时序、对齐、格式/版本信息
│   └── 掩码模式 — 8 种掩码 + 惩罚分评估取最优
└── UI 控制器 — Canvas 渲染、2x 高清 PNG 下载
```

编码引擎暴露为 `QR.generate(text, ecc: 'L'|'M'|'Q'|'H')`，返回 `{ matrix: Int8Array[], size: number, version: number }`。

## 纠错级别说明

| 级别 | 恢复能力 | 适用场景 |
|------|---------|---------|
| L    | ~7%     | 高质量印刷，无遮挡 |
| M    | ~15%    | 常规使用（默认） |
| Q    | ~25%    | 二维码较小或可能部分遮挡 |
| H    | ~30%    | 极端遮挡或磨损场景 |

## 功能介绍

- **版本自动选择** — 根据输入长度和纠错级别自动匹配最优版本（1-40）
- **字节模式** — 完整 UTF-8 编码，支持 CJK 字符和多字节 Unicode
- **掩码优化** — 评估全部 8 种掩码模式，选择惩罚分最低的方案
- **纠错码交错** — 按 QR 规范对数据块和 RS 纠错块进行字节交错
